# LF Gateway

A digital bridge between vision and capital, inspired by Adam Smith's laissez-faire economics.

## About

LF Gateway is a platform that connects Algerian entrepreneurs with investors, fostering innovation without unnecessary barriers. Built with modern web technologies, featuring multilingual support (English, Arabic, French) and responsive design.

## Features

- **For Entrepreneurs**: Submit projects, get expert validation, and connect with premium investors
- **For Investors**: Access curated Algerian startups with transparent data and direct connections
- **Philosophy**: Embracing laissez-faire economics in modern venture capital

## Tech Stack

- HTML5, CSS3 (Tailwind CSS), JavaScript
- Formspree for form handling
- Responsive design with mobile-first approach
- Multilingual support with RTL for Arabic

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser or serve with a local server
3. For development, use `python3 -m http.server` to serve locally
